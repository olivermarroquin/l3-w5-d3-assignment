# Study Goal Tracker

## What the feature does

This app is a simple study goal tracker built with React. Users can add a study goal, choose a difficulty level, edit existing goals, update the difficulty, and delete goals.

The project demonstrates:

- React components
- Props passed between components
- State in a parent component
- Controlled form inputs
- Using submitted form data in the app

## Components

- `App.jsx` stores the main study goal state and handles adding, editing, updating, and deleting goals.
- `GoalForm.jsx` contains the controlled form for adding new study goals.
- `GoalList.jsx` displays the goals and allows users to edit or delete them.

## How to run the app

Install dependencies:

```bash
npm install
```
